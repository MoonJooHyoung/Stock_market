# Stock Market Dividend Calendar

실시간 배당 일정을 관리하는 웹 애플리케이션입니다.

## 🚀 주요 기능

- 📅 **실시간 배당 달력**: Investing.com에서 크롤링한 실제 배당 일정
- 💰 **투자 원금 관리**: 투자 금액 추적 및 관리
- 📊 **보유 종목 관리**: 매수/매도 거래 기록 및 평단가 계산
- 💵 **배당금 관리**: 수동 배당금 기록 및 추적
- 📈 **수익률 계산**: 총 투자금, 배당금, 매도 차익 통합 관리
- 🔄 **자동 업데이트**: 정기적인 배당 데이터 자동 수집
- 🌍 **다국가 지원**: 미국 및 한국 기업 배당 일정

## 🛠️ 기술 스택

- **Backend**: Python Flask, SQLite
- **Frontend**: HTML5, CSS3, JavaScript
- **크롤링**: BeautifulSoup, Requests
- **스케줄링**: Schedule
- **데이터**: Investing.com 크롤링

## 📦 설치 및 실행

### 1. 의존성 설치
```bash
pip install -r requirements.txt
```

### 2. 환경변수 설정 (선택사항)
```bash
cp env_example.txt .env
# .env 파일에서 API 키 설정 (DART API, 공휴일 API)
```

### 3. 애플리케이션 실행
```bash
python run_app.py
```

### 4. 브라우저에서 접속
```
http://localhost:5000
```

## 🔧 사용법

### 배당 데이터 업데이트
1. 웹페이지에서 "데이터 관리" 섹션 열기
2. "배당 데이터 업데이트" 버튼 클릭
3. Investing.com에서 최신 배당 데이터 자동 수집

### 투자 관리
1. **투자 원금 관리**: 투자할 금액 입력
2. **보유 종목 관리**: 매수/매도 거래 기록
3. **배당금 관리**: 수동 배당금 기록

### 달력 보기
- 월별 배당 일정 확인
- 날짜 클릭 시 상세 배당 정보 표시
- 미국/한국 기업 구분 표시

## 🔄 자동 업데이트

### 스케줄러 실행
```bash
python scheduler.py
```

### 스케줄 설정
- **매일 09:00**: 배당 데이터 업데이트
- **매주 월요일 10:00**: 전체 데이터 재수집
- **매시간**: 상태 체크

## 📊 API 엔드포인트

### 달력 데이터
- `GET /api/calendar/{year}/{month}`: 월별 배당 일정

### 크롤링 관리
- `POST /api/crawl/update`: 배당 데이터 수동 업데이트
- `GET /api/crawl/status`: 크롤링 상태 확인
- `GET /api/crawl/companies`: 수집된 기업 목록

### 스케줄러 제어
- `POST /api/scheduler/start`: 자동 업데이트 시작
- `POST /api/scheduler/stop`: 자동 업데이트 중지
- `GET /api/scheduler/status`: 스케줄러 상태 확인
